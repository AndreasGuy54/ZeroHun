                            const getValues = () => {
                                let firstNum = parseInt(document.querySelector('#startValue').value)
                                let lastNum = parseInt(document.querySelector('#endValue').value)

                            // generate numbers from the start value to the end value
                            const generateNumbers = (startVal, endVal) => {
                                let nums = [];

                                for (let i = startVal; i <= endVal; i++) {
                                    nums.push(i);
                                }
                                return nums;
                            }

                            // Display the numbers and make even numbers bold
                            const displayNumbers = (nums) => {
                                let templateRows = '';

                                nums.forEach(num => {
                                    let className = '';

                                    if (num % 2 === 0) {
                                        className = 'even';
                                    } 
                                    
                                    templateRows += `<tr><td class="${className}" >${number}</td></tr>
                                });

                                document.querySelector('#results').innerHTML = templateRows;
                            }

The Code is structured in three major functions.

getValues()

A function that takes in the two input values as numbers; calls the generateNumbers function with the two numbers as arguments and displays a list of generated integers starting from the first number to the second number with the displayNumbers function.

generateNumbers()

generates an array of integers between two numbers using the first parameter as a starting value and the second as the end value.

displayNumbers()

A function that takes in the array of generated intergers and displays each number as table data in a table row. Even numbers within the collection are given a className with styles that make them appear as bold.
